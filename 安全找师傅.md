# 安全找师傅网站

## 1.需求

+ 任务

  实现一个安全的找师傅的网站，这里的师傅包括各种门派的武术师傅，以及各种特别用功的师傅和技师，比如装修，电工，要求增强安全性

+ 基本要求
  1要求实现相关的交流，比如发起武术交流和装修宣传活动。
  2.用户密码采用md5 保存，设器ssl安全链接。
  3.各类师傅和技师可以发布个人的特长信息，客户则可以搜索并且联系相关的师傅。
  4如果有条件，可以实现对师傅的评价、关注。
  5.如有条件，可以实现网上约定价格等功能。
